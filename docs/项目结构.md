# 项目目录与内容放置说明

各目录用途及「谁该把什么放哪里」的约定。**仅以本文档为准**，各模块目录下不再单独放 README。

---

## 根目录一览

```
Tracer_Civilization_Environment/
├── docs/           # 文档
├── src/            # 源代码（.cpp）
├── include/        # 头文件（.h/.hpp）
├── data/           # 静态数据（卡牌/怪物/事件表）
├── assets/         # 美术与界面素材（谁需要谁自己找，放这里）
├── src/main.cpp    # 程序入口
└── *.vcxproj       # 工程文件
```

---

## 一、文档（docs/）

| 放什么 | 放在哪 |
|--------|--------|
| 分工、排期、阶段事项与成果 | docs/课设分工与排期.md |
| MVP、数据表、接口、Mock 约定 | docs/设计与接口.md |
| 项目目录与内容放置说明 | docs/项目结构.md（本文档） |
| B、C 模块设计与接口（数据结构与 API 规格） | docs/BC模块设计与接口.md |

---

## 二、源代码与头文件（按模块）

| 模块 | 负责人 | 源文件放哪里 | 头文件放哪里 |
|------|--------|--------------|--------------|
| 地图 | A | src/MapEngine/ | include/MapEngine/ |
| 战斗引擎 | B | src/BattleEngine/ | include/BattleEngine/ |
| 卡牌系统 | C | src/CardSystem/ | include/CardSystem/ |
| 事件/商店/休息 | D | src/EventEngine/ | include/EventEngine/ |
| 数据层 | E | src/DataLayer/ | include/DataLayer/ |
| 主流程（选节点、调 A/B/D、串联一局） | 指定一人或轮流 | src/GameMain/ | include/GameMain/ |

程序入口：`src/main.cpp`（可只做初始化与调主流程，主流程实现在 GameMain）。

---

## 三、数据（data/）

| 放什么 | 放在哪 |
|--------|--------|
| 卡牌表（id、name、cost、效果等） | data/cards.json 或 data/cards.csv |
| 怪物表（id、name、血量、意图等） | data/monsters.json 或 data/monsters.csv |
| 事件表（id、描述、选项树等） | data/events.json 或 data/events.csv |

字段格式见 docs/设计与接口.md 第二节。由 E 牵头维护，其他人按约定读取。

---

## 四、素材（assets/）

| 放什么 | 放在哪 |
|--------|--------|
| 卡牌图、卡背等 | assets/ 或 assets/cards/ |
| 怪物头像、立绘等 | assets/ 或 assets/monsters/ |
| 界面用图、按钮、背景等 | assets/ 或 assets/ui/ |

约定：谁做到需要素材的模块就自己找，统一放到 assets 下，可自建子文件夹。

---

## 五、小结表（按“内容类型”查放哪）

| 内容类型 | 放置位置 |
|----------|----------|
| 地图相关代码（图、BFS/DFS） | src/MapEngine/ + include/MapEngine/ |
| 战斗相关代码（栈、队列、回合） | src/BattleEngine/ + include/BattleEngine/ |
| 牌堆相关代码（抽弃洗） | src/CardSystem/ + include/CardSystem/ |
| 事件/商店/休息相关代码（树） | src/EventEngine/ + include/EventEngine/ |
| 查表、排序、加载数据 | src/DataLayer/ + include/DataLayer/ |
| 主流程、选节点、串联 | src/GameMain/ + include/GameMain/ |
| 卡牌/怪物/事件表数据 | data/*.json 或 data/*.csv |
| 图片、UI 素材 | assets/（可分子目录） |
| 课设文档 | docs/ |
