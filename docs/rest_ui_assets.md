# 休息界面（仿 StS）所需素材清单

休息 UI 已按「杀戮尖塔」休息站风格实现：中央「要做些什么？」+ 左右两个大按钮（**回血** / **升级一张牌**）。以下素材均为**可选**，未提供时用纯色按钮 + 文字显示。

---

## 1. 背景图（可选）

| 路径 | 说明 | 建议尺寸 |
|------|------|----------|
| `assets/backgrounds/rest_bg_cn.png` | 休息站专用背景（营地/篝火/洞窟等） | 与窗口一致或 1920×1080 |

- 若不提供，会回退使用事件背景 `assets/backgrounds/event_bg_cn.png`。
- 风格建议：国风、偏暗、可带篝火/营火、石地等，与事件背景统一。

---

## 2. 休息/回血按钮图标（可选）

| 路径 | 说明 | 建议尺寸 |
|------|------|----------|
| `assets/ui/rest/icon_heal.png` | 左侧「回血」大按钮上的图标 | 64×64 或 128×128（透明底） |

- 内容建议：床+枕头、或心形+治疗、或 Zzz 睡眠符号等，能一眼看出「休息/回血」。
- 若不做，界面会只显示文字「回血 (+N)」，按钮为绿色块。

---

## 3. 锻冶/升级按钮图标（可选）

| 路径 | 说明 | 建议尺寸 |
|------|------|----------|
| `assets/ui/rest/icon_smith.png` | 右侧「升级一张牌」大按钮上的图标 | 64×64 或 128×128（透明底） |

- 内容建议：铁砧+锤子、或锻冶火花、或「↑」升级箭头等，能一眼看出「升级」。
- 若不做，界面会只显示文字「升级一张牌」，按钮为暗红色块。

---

## 4. 当前已实现、无需额外素材的部分

- **双大按钮布局**：左绿（回血）、右红褐（升级），带描边与 hover 变色。
- **中央标题**：「要做些什么？」居中显示。
- **升级选牌**：点击「升级一张牌」后进入选牌列表，带「← 返回」回到双按钮。
- **字体**：与事件/商店共用现有字体。

---

## 5. 目录结构建议

```
assets/
  backgrounds/
    rest_bg_cn.png     # 休息专用背景（可选）
  ui/
    rest/
      icon_heal.png    # 回血图标（可选）
      icon_smith.png   # 锻冶/升级图标（可选）
```

把对应文件放到上述路径后，程序会自动加载；缺文件时不影响运行，仅无图显示。
