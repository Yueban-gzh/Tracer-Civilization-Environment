背景音乐说明
============

扫描目录：assets/music/（不递归）

支持的扩展名：.mp4、.mp3、.ogg、.flac、.wav  
（SFML 对部分 .mp4 容器可能无法解码；若无法播放请改用 .ogg 或 .mp3。）

行为
----
- 地图 / 事件 / 商店 / 休息 / 宝箱等非战斗流程：循环播放**一首**「地图曲」，在这些界面之间**不会重复切歌或重头播放**（已在播则保持）。
- 每场战斗开始时：从对应**战斗曲池**中**均匀随机**一首（在池内随机起点后顺序尝试解码，成功即播），循环播放；**退出战斗**时再切回地图曲。
- **普通战斗 / 精英**（Enemy、Elite）：从「普通战斗池」随机（见下）。
- **最终 Boss 节点**（地图上的 Boss）：从「Boss 战斗池」随机；若该池为空则退回普通战斗池。

战斗曲命名（与地图曲互斥：地图曲不会进战斗池）
------------------------------------------
- **Boss 专用**（仅 Boss 节点战会播）：文件名（不含扩展名）以 **`boss_bgm`** 开头，ASCII 不区分大小写。  
  例：`boss_bgm_finale.mp3`、`Boss_BGM_Phase2.ogg`
- **普通 / 精英战**：`assets/music` 下其余可播放文件（去掉地图曲、且不以 `boss_bgm` 开头的文件）。  
  若你希望某首只给 Boss 用，请改名为 `boss_bgm_...`；旧资源里文件名含「boss战」等中文**不会**自动进 Boss 池，需自行改名或复制一份并加 `boss_bgm` 前缀。
- 若目录里**只有** `boss_bgm*` 文件、没有普通曲：程序会把**整池**当作普通战兜底，避免小怪战无声。

地图曲（按优先级选用一首）
--------------------------
1. 文件名（不含扩展名）为 map_bgm（大小写不敏感），如 map_bgm.mp3
2. 文件名为 ambient 或 explore
3. 文件名中含「圣森」二字（如现有资源「…圣森.mp3」）
4. 若以上都没有：按路径排序后的**第一首**文件作为地图曲；战斗随机池为其余文件。

路径示例
--------
  assets/music/map_bgm.ogg
  assets/music/沉溺于空虚之中 - 圣森.mp3

若目录下没有可播放文件，游戏静默无背景音乐。
